# Carry Helper

> Note: This mod is still actively under development, so not all features may be fully fletched out.

The Dungeon Hub Carry Helper (or simply **Carry Helper**) is a Hypixel SkyBlock utility mod designed to streamline the carry workflow for Dungeon Hub carriers.

By integrating directly with the Dungeon Hub backend, which our discord bot uses as well, the mod provides convenient in-game access to carry information and automates repetitive tasks, allowing you to focus on completing carries instead of managing them manually.

### Features

* View your currently claimed carry tickets directly in-game.
* Send a message containing a clickable button that instantly invites the customer to your party.
* Automatically log completed carries to the Dungeon Hub backend, eliminating the need to manually submit completion information through Discord.
* *(Coming soon)* Receive notifications when new tickets become available for claiming.

Carry Helper is built to make the carrying process faster, smoother, and more efficient for Dungeon Hub carriers.


<details>
<summary>Privacy & Network Usage</summary>

To provide certain features, Carry Helper communicates with the Dungeon Hub API at:

https://api.dungeon-hub.net/

This includes functionality such as:

* Automatic carry logging
* Fetching available tickets
* Fetching claimed tickets

Access to the API requires authentication through Discord. Discord authentication is used solely to verify your Discord identity and obtain your Discord ID, which is used by the Dungeon Hub backend to determine permissions and access to carrier-related data.

Carry Helper does not access your Discord account, messages, contacts, or any other Discord data beyond what is required for authentication.

No sensitive or private data is sent to third-party services as part of using this mod.
